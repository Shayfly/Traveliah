# Travelia — גרסת Super Pro

אתר חיפוש טיסות וחופשות עם עיצוב מודרני, תמיכה בעברית ואנגלית, וממשק משתמש חכם.

## איך מריצים מקומית

```bash
npm install
npm run dev
```

## איך בונים ומעלים ל-GitHub Pages

```bash
npm run build
npm run deploy
```

ודא שקובץ vite.config.js כולל:
```js
base: '/traveliah/'
```
